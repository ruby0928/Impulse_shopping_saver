# 💸 購物慾望與股票換算器 (Shopping Impulse Saver)

這是發想自一篇 threads 貼文的理財輔助工具。透過將消費金額即時換算為等值的台股標的（如台積電、0050），並計算其複利價值，幫助使用者在衝動購物前重新評估金錢的「機會成本」，並記錄每一次的理財決定。

---

## ✨ 功能介紹

### 💱 即時換算
輸入想買的物品與金額，選擇股票標的（台積電 / 0050），立即換算出可買到的股數與複利預估值。

### 🤔 決策紀錄
換算後提供三個選項，讓每次消費衝動都留下紀錄：

| 選項 | 說明 |
|---|---|
| 😤 我需要它，買了！ | 承認購買，不留紀錄 |
| 📈 我選擇投資股票 | 紀錄股數與成本價，每次開啟自動抓取當日股價計算損益 |
| 🐷 存起來 | 累計節省金額，看看你到底省了多少錢 |
| 只是算好玩的，不紀錄 | 關閉選項，不存任何資料 |

### 📊 損益追蹤
- 投資紀錄自動顯示現價、損益金額與百分比
- 頁面右上角顯示所有投資紀錄的合計損益（紅/綠）
- 支援單筆刪除紀錄

### 🐷 存款累計
省下來的每一筆錢都會列表顯示，右上角即時加總「累計節省」金額。

---

## 🚀 技術亮點

### 即時金融數據串接
- 透過 **FinMind API** 串接台股收盤價
- 內建智能日期判定：自動回推至最近的交易日，避免週末或盤前無資料

### 前端快取優化
- **LocalStorage 1 小時快取：** 股價資料暫存於瀏覽器，重複查詢同一標的時秒出結果，節省 API 配額
- **決策紀錄持久化：** 投資與存款紀錄寫入 localStorage，重新整理後仍保留

### 機會成本視覺化
- 內建複利公式，以年化報酬率 7% 計算 3 年後預估增值
